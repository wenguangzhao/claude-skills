📦 shadcn Registryとは？

shadcn/uiは、コンポーネントをインストールして使用できる革新的なシステムを提供しています。npmパッケージとして配布するのではなく、コードを直接プロジェクトにコピーする方式を採用。これにより完全なカスタマイズが可能です。

さらに、Registry Systemにより、shadcn公式コンポーネントだけでなく、サードパーティが提供する高品質なコンポーネントライブラリも同じCLIコマンドで簡単にインストールできます：

// 公式コンポーネント npx shadcn add button // サードパーティレジストリ npx shadcn add @magicui/animated-beam

追加の設定やインストールは一切不要。1コマンドでコンポーネントがプロジェクトに追加され、すぐにカスタマイズして使用できます。
